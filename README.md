### Manual installation


**right-click**Garry's Mod - **properties..** - startup options“**-condebug**”
![](https://static.truegear.tech/GitImg/GMod/img1.png)

### VRMod_TrueGear

Click   [VRMod_TrueGear](https://steamcommunity.com/sharedfiles/filedetails/?id=3609522389)   installation

### Development

Just output [TrueGear] :{Effect Name} will trigger the corresponding effect

Such as:   print("[TrueGear] :{PickUpItem}")   can be triggered.
![](https://static.truegear.tech/GitImg/GMod/img2.png)

（If you need special effects that do not exist in the list, you can contact the administrator to add）
